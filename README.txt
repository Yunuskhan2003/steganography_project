Steganography Project
----------------------
This project demonstrates how to hide secret messages inside images using Python and the Pillow library.

Files:
- encoder.py: To encode (hide) a message in an image.
- decoder.py: To decode (extract) a hidden message from an image.

Usage:
1. Place an image named 'original_image.png' in the folder.
2. Use encode_image() to hide your message.
3. Use decode_image() to retrieve it from 'encoded_image.png'.

Ensure you have Pillow installed:
pip install pillow
